Electron: Pros & Cons

Back to list

Pros

  • Only have to support one browser (Chromium)
  • App runs in standalone browser instance, which helps to keep it secure
  • App can interact with native APIs to provide such things as native notifications
  • All app files are local so they will always be available, even offline (however this is not the case for API data)
  • Can be easily packaged for individual clients, giving specific functionality to each

Cons

  • Any change to the codebase will require an update on the user's machine
  • Will cause a separation of UI and API (Swagger will no longer sit alongside UI)
  • May have implications if we want more mobile functionality - a desktop app, yet a mobile website. Might look strange
